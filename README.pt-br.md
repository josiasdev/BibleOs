# BibleOS 📖🐧

> 🌍 **Traduções:** [Click here to read the English version](README.md)

O BibleOS é uma distribuição Linux avançada, estável e leve baseada no **Linux Mint XFCE**. Foi meticulosamente projetada para fornecer um ambiente de trabalho digital cristão completo, feito sob medida para igrejas, ministérios, técnicos de áudio/vídeo e estudos teológicos individuais. 

Ao combinar a estabilidade robusta do Linux Mint com softwares eclesiásticos pré-configurados, codecs multimídia e um layout familiar para usuários do Windows, o BibleOS elimina a dificuldade e o tempo gasto para configurar a infraestrutura digital de um ministério do zero.

---

## 🌟 Principais Funcionalidades

- **Escrituras Sagradas Integradas:** Inclui traduções de domínio público com texto completo pré-instaladas localmente (ex: *Almeida Corrigida Fiel - ACF* para o Português, *Authorized King James Version - KJV* para o Inglês) através do ecossistema SWORD.
- **Pronto para Projeção em Cultos:** Vem pré-configurado com **Holyrics** e excelentes motores de apresentação de código aberto.
- **Controle de Mesa de Som Digital:** Aplicativos e ferramentas dedicadas para controlar mixers de áudio digitais (ex: Behringer, Soundcraft, Yamaha) prontos para uso, oferecendo uma alternativa nativa e fluida aos setups de Windows.
- **Ecossistema de Ferramentas Mint:** Integração total dos utilitários do Linux Mint (`mintupdate`, `mintinstall`, `mintbackup`) e suporte multimídia completo e nativo (`mint-meta-codecs`).
- **Otimização do XFCE:** Ambiente altamente polido e leve, com temas visuais intuitivos para quem está migrando do Windows, garantindo alto desempenho tanto em equipamentos modernos quanto em computadores mais antigos das igrejas.

## 🛠️ Stack de Tecnologias e Dependências

- **Sistema Base:** Linux Mint XFCE
- **Motor de Customização da ISO:** Cubic (Custom Ubuntu ISO Creator)
- **Framework Bíblico:** SWORD Project API / Xiphos / BibleTime
- **Camadas de Execução:** Wine (Otimizado para softwares multimídia de igreja) e OpenJDK Java
